? SIGNAGE OS 사용법 안내 v4.0
현재 관리자 URL:
① 매장 + 사이니지 등록
1매장 관리 → + 매장 추가
2매장 저장 후 행 클릭하면 사이니지 목록 펼쳐짐
3+ 사이니지 추가 → 이름/위치 입력 → 추가
4사이니지별 [📋 URL복사] → 태블릿에 붙여넣기
💡 사이니지마다 다른 URL = 개별 콘텐츠 재생!
② 콘텐츠 등록
1콘텐츠 관리 → 매장 탭 선택 → + 콘텐츠 등록
2유형 선택 (이미지/영상/유튜브/웹)
3URL 입력 (구글드라이브 링크 자동 변환)
💡 이미지: imgur.com 업로드 후 URL
💡 영상: 유튜브 업로드 후 URL
③ 플레이리스트 설정
1플레이리스트 → + 플레이리스트 생성
2매장 선택 → 사이니지 선택 (개별 지정!)
3재생할 콘텐츠 체크 → 생성
💡 사이니지마다 다른 플레이리스트 가능!
④ 원격 제어
1실시간 관제 → 매장별 사이니지 카드
2각 카드에서 직접 제어
