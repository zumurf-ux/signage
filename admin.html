관리자 URL:
① 매장 등록
1매장 관리 → + 매장 추가
2매장명, 지역, 담당자, 연락처 입력
3총 운영시간 설정 (예: 09:00 ~ 22:00)
4스케줄 추가 — 이름 선택 + 시간 입력
예) 점심: 11:30~14:00
5저장 클릭
② 사이니지 등록
1매장 관리 → 매장 행 클릭 → 펼쳐짐
2이름(예: 입구), 위치(예: 1층 입구) 입력
3화면 방향 선택 → + 추가
4📋 URL복사 → 태블릿 Chrome에 붙여넣기
💡 사이니지마다 고유 URL로 개별 관리!
③ 콘텐츠 등록
1콘텐츠 관리 → 매장 탭 선택
2+ 콘텐츠 등록 → 이름, 유형 선택
3URL 입력 → 등록
🖼 이미지: imgur.com → New post → 업로드
→ 이미지 우클릭 → 이미지 주소 복사
▶ 영상: 유튜브 URL 그대로 붙여넣기
④ 플레이리스트 설정
1플레이리스트 → 매장 클릭 → 펼치기
2사이니지 카드 하단 + 추가 클릭
3이름 입력 → 스케줄 선택
4콘텐츠 체크 → 생성
💡 스케줄별 자동 전환!
점심 시간 → 점심 플레이리스트 자동 재생
⑤ 원격 제어
1실시간 관제 → 매장별 사이니지 카드
⟳ 새로고침 — 화면 새로고침
▶ 재시작 — 플레이어 재시작
⏸ 대기화면 — 시계 화면으로 전환
↺ 재생재개 — 콘텐츠 재생 재개
※ 명령은 약 5초 내 적용
⑥ 태블릿 전체화면
1태블릿에서 URL 접속 후 화면 터치
2자동으로 전체화면 전환
3Chrome → 홈 화면에 추가 권장
💡 홈 화면 아이콘 실행 시
브라우저 UI 없이 전체화면!
